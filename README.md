# 麻雀点数計算学習アプリケーション

麻雀の点数計算（符・飜数・点数）をクイズ形式で学習できるWebアプリケーションです。

## 技術スタック

- **フロントエンド**: React 19 + TypeScript
- **ビルドツール**: Vite
- **スタイリング**: Tailwind CSS
- **テスト**: Vitest + fast-check + React Testing Library

## プロジェクト構造

```
src/
├── components/     # UIコンポーネント
├── domain/         # ドメインロジック（点数計算など）
├── services/       # サービス層（問題生成、クイズ管理など）
├── types/          # TypeScript型定義
└── test/           # テスト関連ファイル
```

## セットアップ

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで http://localhost:5173/ を開きます。

### ビルド

```bash
npm run build
```

### テストの実行

```bash
# 全テストを実行
npm test

# ウォッチモードでテストを実行
npm run test:watch

# UIモードでテストを実行
npm run test:ui
```

## 開発ガイドライン

### テスト戦略

このプロジェクトでは2種類のテストを使用します：

1. **ユニットテスト**: 個々の関数やコンポーネントの動作を検証
2. **プロパティベーステスト**: fast-checkを使用して、すべての入力に対して成立すべき性質を検証

各プロパティベーステストは最低100回の反復を実行します。

### コーディング規約

- すべてのコードはTypeScriptで記述
- コンポーネントは関数コンポーネントを使用
- ビジネスロジックとUI層を明確に分離
- 日本語のコメントと変数名（必要に応じて）を使用
